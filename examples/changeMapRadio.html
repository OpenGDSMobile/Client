OpenGDS Mobile Client Example

지도 변경 (라디오 타입)

지도 시각화 후 eGovFrameUI를 통해 간단하게 배경지도를 변경 가능한 라디오 버튼 타입의 사용자 인터페이스 생성 가능.
제공 지도 : OpenStreetMap, VWorld

                        var ui = new OGDSM.eGovFrameUI();

                        ui.baseMapRadioBox(obj, 'mapSelect', 'OSM VWorld VWorld_m VWorld_s VWorld_g');
