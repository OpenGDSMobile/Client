풀 스크린 지도

  화면 전체를 지도로 채우고 사이즈가 변경이되어도 자동으로 지도 해상도가 맞춰지는 지도 시각화

지도 변경(라디오 버튼 타입)

  라디오 버튼 타입으로 배경지도를 변경하는 사용자 인터페이스(OSM, VWowlrd 제공)

지도 변경(셀렉트 버튼 타입)

  셀렉트 버튼 타입으로 배경지도를 변경하는 사용자 인터페이스(OSM, VWowlrd 제공)

브이월드 데이터 시각화

  국내 공간정보 오픈플랫폼인 브이월드 WMS 데이터 시각화를 위한 인터페이스 및 방법 제공 (총 72종 데이터 제공)

레이어 관리 기능

  시각화한 레이어 관리 사용자 인터페이스 생성(삭제, 투명도, 시각화 여부 가능)

UX 컴포넌트 자동 생성

  전자정부 모바일 표준 프레임워크 실행환경에서 제공하고 있는 UX 컴포넌트 자동 생성 (버튼, 셀렉트, 스위치, 라디오)

벡터 레이어 시각화

  GeoServer 기반 벡터 데이터 시각화
(Point, Polygon, Line)

서울 열린 데이터 광장

  서울 열린 데이터 광장 데이터 중 실시간 환경 데이터 시각화 (막대 차트, 벡터 지도)

공공데이터 포털

  공공데이터 포털에서 제공되고 있는 AirKorea의 전국 실시간 환경 데이터 시각화 (막대 차트, 벡터 지도)

속성정보 시각화

  벡터 데이터 시각화시 속성정보 동시 시각화

GeoServer 레이어 리스트

  GeoServer에 등록된 특정 워크스페이스의 레이어 리스트 받아오기
