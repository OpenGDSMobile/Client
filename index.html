OpenGDS Mobile Client Library 1.2

활용 오픈소스 리스트

  1. 전자정부 모바일 표준 프레임워크 실행환경
    • jQuery 1.9.1
    • jQuery Mobile 1.3.2
  2. OpenLayers 3.5
  3. Proj4js
  4. D3 3.5.5(Data-Driven Documents)

기본 세팅 (실행환경만 사용할 경우)

  1. 전자정부 모바일 표준 프레임워크 실행환경 다운 후 WebApp 폴더 내부만 따로 분리 (다운로드 링크)
  2. OpenLayers3 자바스크립트 /CSS 파일 다운로드 후 모바일 실행환경 안 js / CSS 폴더에 ol3 폴더 생성 후 복사 (다운로드 링크)
  3. D3.js 자바스크립트 /CSS 파일 다운로드 후 모바일 실행환경 안 js / CSS 폴더에 d3 폴더 생성 후 복사 (다운로드 링크)
  4. Proj4js 자바스크립트 파일 다운로드 후 모바일 실행환경 안 js 폴더에 proj4js 폴더 생성 후 복사 (다운로드 링크)
  5. OpenGDSMobileLib 폴더를 다운 받은 실행환경 안 js 폴더에 복사
  6. 메인 페이지에(ex : index.html) 생성 후 아래와 같이 작성하면 지도 시각화 가능

기본 세팅 (전자정부 표준 프레임워크 프로젝트 생성)

  1. OpenLayers3 자바스크립트 /CSS 파일 다운로드 후 모바일 실행환경 안 js / CSS 폴더에 ol3 폴더 생성 후 복사 (다운로드 링크)
  2. D3.js 자바스크립트 /CSS 파일 다운로드 후 모바일 실행환경 안 js / CSS 폴더에 d3 폴더 생성 후 복사 (다운로드 링크)
  3. Proj4js 자바스크립트 파일 다운로드 후 모바일 실행환경 안 js 폴더에 proj4js 폴더 생성 후 복사 (다운로드 링크)
  4. OpenGDSMobileLib 폴더를 다운 받은 실행환경 안 js 폴더에 복사
