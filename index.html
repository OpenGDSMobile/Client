오픈소스 기반 OpenGDS Mobile Client 라이브러리

(OpenGDS Mobile Client Library based on Open Source)

OpenGDS Mobile Client는 공간정보와 시각화를 주제로 하고 있는 자바스크립트 라이브러리로 여러 오픈소스를 활용하여 이를 편리하게 사용할 수 있는 기능 및 HTML5 API를 포함하고 있다.

사용된 오픈소스(Usage Open Source)

  • 표준프레임워크 모바일 실행환경 (eGovframework in Korea)
  • 오픈레이어 3 (OpenLayers 3)
  • D3.js
  • 부트스트랩 (BootStrap)

단위 기능 예제(Unit Examples)

  • 배경 지도 시각화
  • 벡터 레이어 시각화 *(GeoServer)
  • 벡터 레이어 다중 레이어 관리 * (GeoServer)
  • 벡터 레이어 스타일 변경 * (GeoServer)
  • 속성정보 시각화 *(Application Server / GeoServer)
  • 속성정보 오프라인 편집 *(Application Server / GeoServer)
  • 공공 데이터 차트 시각화 *(Application Server)
  • VWorld Open API 시각화 - WFS
  • VWorld Open API 시각화 - WMS
  • HTML5 API IndexedDB 저장소 관리
  • HTML5 API WebSocket 활용 예제
※ 예제는 ApplicationServer 또는 GeoServer가 반드시 필요합니다.

통합 예제(Intergated Examples)

  • 공간 속성정보 오프라인 및 실시간 편집
  • 수집 공공데이터 시각화
